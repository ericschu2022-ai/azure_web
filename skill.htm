涵蓋從資料庫管理、程式開發到雲端自動化部署的完整技術棧。

SQL MS SQL Server Analysis Services (SSAS)

負責商業智慧的多維度數據建模與資料分析，提供高效的查詢效能。

Reporting MS SQL Server Reporting Services (SSRS)

設計與發佈各類企業級報表，實現數據視覺化與自動化發送。

熟練 SQL 語法，並具備數據清洗、轉換、載入 (ETL) 的實務經驗。

Dev Python 基礎語法

運用 Python 進行自動化腳本撰寫、數據處理及網頁爬蟲應用。

BI Power BI 與 DAX 語法

精通 Power BI 建模與複雜 DAX 公式編寫，將數據轉化為決策建議。

DevOps MS Azure 與 GitHub CI/CD

具備雲端環境部署經驗，並能透過 GitHub 設定自動化集成與發佈流程。
